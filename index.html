About

A WebDNA console over the Linode (Akamai Cloud) API, backed by a provider-agnostic client / asset store. Each resource screen reads live from the API; the client / asset linkage lives in db/.

Asset model

Each Linode, bucket, domain and firewall is an asset carrying a client, a provider cost and a renewal/status — a provider-agnostic model, so a new provider or resource type is a new adapter + type, not a schema change.
